# 01 - Two Sum

## Problem Description
Given an array of integers `nums` and an integer `target`, return indices of the two numbers such that they add up to `target`.

## Approach
Used a **Hash Map** to store values and their indices. This allows us to find the complement in $O(1)$ time.

## Complexity
- **Time Complexity:** $O(n)$
- **Space Complexity:** $O(n)$
